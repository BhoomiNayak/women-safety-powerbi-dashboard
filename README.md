# Women Safety in India — Crime Investigation & Justice Gap Analysis

## About
An interactive Power BI dashboard analyzing NCRB 2023 data on crimes 
against women across all 36 Indian states and Union Territories.

## Dashboard Highlights
- 448K total cases reported in 2023
- Justice Gap analysis by state
- Chargesheeting rate vs crime volume scatter plot
- State wise crime distribution map

## Tools Used
- Power BI Desktop
- Python (Pandas) for data cleaning
- Data Source: NCRB Crime in India 2023

## Key Findings
- Uttar Pradesh accounts for 15% of national cases
- Manipur has 92.7% pendency rate — highest in India
- Mizoram leads with 97.8% chargesheeting rate
- 32.47% of cases remain pending at year end

## Data Source
National Crime Records Bureau (NCRB) — Crime in India 2023
https://ncrb.gov.in
